# 仿安卓网易云音乐，微信小程序

### 功能特性

1. 头条轮播
2. 首页下拉，推荐歌单刷新
3. 搜索，热搜榜
4. 单曲列表，上拉刷新增加歌曲
5. 播放界面（待完善）

### 安装前提

该微信小程序只作为个人使用，不能进行其他商业用途。该小程序所需接口借助于[**NeteaseCloudMusicApi**](https://github.com/Binaryify/NeteaseCloudMusicApi)，本地运行该nodejs才可以继续使用本小程序所有功能。
